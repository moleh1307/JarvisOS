# Project Index

## Purpose

Use this as the first project-routing note for fresh chats. It should help the agent pick the right project note quickly without reading the whole vault.

## Read Strategy

1. Identify the project or domain from the user's request.
2. Read the matching project note below.
3. Read the listed latest session/handoff only if needed.
4. Do not scan unrelated projects by default.
